CRYPTO TRADING BOT 2025 - QUICK INSTALLATION STEPS
==================================================

STEP 1: Install Prerequisites
-----------------------------
1. Ensure Python 3.9+ is installed
2. Open Command Prompt as Administrator:
   - Press Windows Key + X
   - Select "Terminal (Admin)" or "Command Prompt (Admin)"
   - Click "Yes" on the UAC prompt

STEP 2: Enable WSL2 (Recommended Setup)
---------------------------------------
Run these commands one by one in Administrator Command Prompt:

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

STEP 3: Restart Your Computer
-----------------------------
This is MANDATORY after enabling WSL features. Restart now.

STEP 4: After Restart - Install Ubuntu
--------------------------------------
1. Open Command Prompt as Administrator again
2. Run: wsl --install -d Ubuntu
3. Wait for download and installation
4. Create username and password when prompted

STEP 5: Set Up Trading Bot in WSL
---------------------------------
1. Open Ubuntu from Start Menu
2. Navigate to project directory:

cd /mnt/c/dev/tools/crypto-trading-bot-2025/

3. Install Python dependencies:

python -m pip install --upgrade pip
pip install -r requirements.txt

4. Configure API credentials:

cp config.json config.json.backup
# Edit config.json with your Kraken API key and secret

STEP 6: Test Installation
-------------------------
Run these commands to verify everything works:

# Test API connection
python scripts/test_kraken_connection.py

# Check bot components
python scripts/check_bot_status.py

STEP 7: Start Trading
---------------------
# For safe testing (recommended first):
python start_paper_trading.py

# For live trading (after testing):
python main.py

ALTERNATIVE: Windows Native Installation
=========================================
If you prefer not to use WSL:

1. Open Command Prompt in project directory:
cd /d C:\dev\tools\crypto-trading-bot-2025

2. Install dependencies:
python -m pip install --upgrade pip
pip install -r requirements.txt

3. Configure API keys in config.json

4. Run: START_TRADING_BOT.bat

DONE! Your trading bot is ready to use.

Project location: /mnt/c/dev/tools/crypto-trading-bot-2025/
